Visceral Design

First impressions and emotional responses through visual design, typography, and color schemes

Design Philosophy

Visceral design focuses on the immediate emotional response users have when they first encounter the international student platform. The design decisions prioritize warmth, inclusivity, and cultural sensitivity to create a welcoming environment for students from diverse backgrounds.

This approach supports the client's goal of creating an inclusive platform and addresses user stories like "As a new international student, I want to feel welcomed and supported so that I can confidently navigate my new environment."

Page Layouts

Wireframes demonstrate the visual hierarchy and layout structure for key pages. These layouts prioritize clarity and accessibility while maintaining cultural inclusivity.
