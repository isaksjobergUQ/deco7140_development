Detailed Usage Reflection

1. Problem Understanding and Research

My Independent Work: I spent significant time analyzing the client brief, reviewing lecture materials on UX design principles, accessibility standards (WCAG 2.2), and ethical design frameworks. I conducted thorough research on international student needs and created detailed user personas based on my understanding of the target audience.

Tools Used: Lecture notes, course readings, personal research, and independent analysis of user needs and client requirements.

2. Design and Planning

My Independent Work: I created comprehensive wireframes in Miro, designed visual elements and layouts in Figma, and developed the overall information architecture. I planned the site structure, navigation patterns, and user flow based on my research findings.

Tools Used: Miro for wireframing, Figma for visual design, and my own design thinking process based on UX principles learned in lectures.

3. Implementation and Coding

Collaborative Process: I used Cursor and ChatGPT to implement my design ideas into functional code. This involved translating my wireframes into HTML structure, creating CSS for styling, and developing JavaScript for interactive elements.

My Role: I provided clear specifications, reviewed all generated code, tested functionality, and made necessary adaptations to ensure the code met my design requirements and coding standards.

4. Quality Assurance and Refinement

My Independent Work: I tested all functionality, verified accessibility compliance, checked responsive design across devices, and refined the user experience based on my understanding of the target audience.

Verification Process: I cross-referenced AI suggestions with course materials, tested code manually, and ensured all implementations aligned with my original design vision and user research findings.

Specific AI Usage Areas

HTML Structure

Used AI to generate semantic HTML structure based on my wireframes and design specifications. I provided detailed requirements and reviewed all generated code for accessibility and semantic correctness.

CSS Styling

Collaborated with AI to implement responsive CSS layouts, color schemes, and styling that matched my design vision. I specified exact requirements and adapted suggestions to fit my design system.

JavaScript Functionality

Used AI to develop interactive features like accordions, image viewers, and form handling. I provided clear specifications and tested all functionality to ensure it met my requirements.

Code Optimization

Collaborated with AI to optimize code structure, improve performance, and ensure best practices. I reviewed all suggestions and implemented only those that aligned with my coding standards.
