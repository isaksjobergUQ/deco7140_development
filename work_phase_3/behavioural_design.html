Behavioural Design

Interactive components and user experience design that supports effective task completion and user satisfaction

Design Philosophy

Behavioural design focuses on how users interact with the platform and complete tasks effectively. This level addresses usability, functionality, and the overall user experience to ensure international students can easily navigate and accomplish their goals.

This approach supports client needs for an intuitive platform and addresses user stories like "As an international student, I want to quickly find relevant tips and resources so that I can adapt to my new environment efficiently."

Button Styles

Interactive buttons demonstrate clear affordances, visual feedback, and accessibility features that support effective user interaction on the international student platform.

Primary Button

Primary buttons use strong color contrast and clear action labels to stand out as the main call-to-action. The hover and pressed states provide immediate visual feedback, supporting user confidence through clear signifiers of interactivity.

Secondary Button

Secondary buttons provide alternative actions with subtle styling that doesn't compete with primary buttons. The design maintains consistency while offering clear visual hierarchy and accessibility through proper contrast ratios.

Icon Button

Icon-only buttons use clear visual symbols with proper ARIA labels for screen readers. The design maintains consistency with other buttons while providing intuitive visual cues for common actions like search, close, and share.

Call-to-Action Button

Call-to-action buttons use compelling language and enhanced visual prominence to encourage user engagement. The design emphasizes urgency and value while maintaining accessibility standards and providing clear feedback states.
